Walkthroughs and whitepapers

The following blogs use a case study of sentiment prediction for a movie review to illustrate the process of executing a complete machine learning workflow. This includes data preparation, monitoring Spark jobs, and training and deploying a ML model to get predictions directly from your Studio notebook.

  • Create and manage Amazon EMR clusters from SageMaker Studio to run interactive Spark and ML workloads.

  • To extend the use case to a cross-account configuration where SageMaker Studio and your Amazon EMR cluster are deployed in separete AWS accounts, see Create and manage Amazon EMR clusters from SageMaker Studio to run interactive Spark and ML workloads - Part 2.
