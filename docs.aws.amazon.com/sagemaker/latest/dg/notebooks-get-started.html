Get Started

To get started, you or your organization's administrator need to complete the Amazon SageMaker Studio onboarding process. For more information, see Onboard to Amazon SageMaker Domain.

You can access a Studio notebook in any of the following ways:

  • You receive an email invitation to access Studio through your organization's IAM Identity Center, which includes a direct link to login to Studio without having to use the Amazon SageMaker console. You can proceed to the Next Steps.

  • You receive a link to a shared Studio notebook, which includes a direct link to log in to Studio without having to use the SageMaker console. You can proceed to the Next Steps.

  • You onboard to Studio and then log in to the SageMaker console. For more information, see Onboard to Amazon SageMaker Domain.

Launch Amazon SageMaker

Complete the steps in Launch Amazon SageMaker Studio to launch Studio.

Next Steps

Now that you're in Studio, you can try any of the following options:

  • To create a Studio notebook or explore Studio end-to-end tutorial notebooks – See Amazon SageMaker Studio Tour in the next section.

  • To familiarize yourself with the Studio interface – See Amazon SageMaker Studio UI Overview or try the Getting started notebook by selecting Open the Getting started notebook in the Quick actions section of the Studio Home page.
