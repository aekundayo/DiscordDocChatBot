Available Amazon SageMaker Images

This page lists the SageMaker images and associated kernels that are available in Amazon SageMaker Studio. SageMaker images contain the latest Amazon SageMaker Python SDK and the latest version of the kernel. For more information, see Deep Learning Containers Images.

The following table lists the image ARN format for each Region. To create the full ARN for an image, replace the resource-identifier placeholder with the corresponding resource identifier for the image from the SageMaker images and kernels table.

Note

SageMaker Distribution images use a distinct set of image ARNs.
